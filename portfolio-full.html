Hebrew Bible Analysis Suite

Comprehensive browser-based platform for computational analysis of the Hebrew Bible (Tanakh), featuring 11 interactive research tools — including the first open-source JavaScript replication of the landmark 1994 Witztum-Rips-Rosenberg equidistant letter sequence experiment.

Data Pipeline: Processed 304,805 consonantal Hebrew letters with character-level metadata across 39 Tanakh books. Built compression pipeline (630MB raw → 21MB served) using gzip via DecompressionStream API. Pre-computed 1.74M ELS hits for Hebrew calendar date heatmap.

Technical Architecture: Web Workers for off-thread computation (ELS scan + WRR permutation testing), KMP + Boyer-Moore pattern matching with Hebrew final-form (sofit) letter normalization, custom GIF89a encoder with LZW compression, Three.js 3D WebGL visualization with auto-rotate and video capture. Zero npm packages, zero build steps. PWA — works offline after first visit.

Statistical Rigor: WRR replication achieved P = 0.0012 (1 in 840) vs. original P < 0.00002. No independent researcher has reproduced WRR's original P-value in 30+ years — the MBBK critique (Statistical Science, 1999), the Hebrew University Aumann Committee replications, and this implementation all point to the same conclusion. The ~75x gap is fully documented with transparent methodology.

AI-Assisted Development: Built with Claude Code (Anthropic) as a development accelerator for code generation, debugging, and algorithmic implementation.
